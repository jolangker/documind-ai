## Tech Stack

This application is built with a modern, robust technology stack:

- **Framework**: [Nuxt 4](https://nuxt.com) - The Intuitive Vue Framework.
- **UI Library**: [Nuxt UI](https://ui.nuxt.com) - A comprehensive UI library for Nuxt.
- **Database & Auth**: [Supabase](https://supabase.com) - Open Source Firebase alternative for database and authentication.
- **AI Integration**: [Vercel AI SDK](https://sdk.vercel.ai) - The TypeScript library for building AI-powered applications.
- **LLM Provider**: [OpenAI](https://openai.com) - Powering the intelligence behind the chat.
- **PDF Processing**: [UnPDF](https://github.com/unjs/unpdf) & [LangChain](https://js.langchain.com) - For efficient PDF parsing and text chunking.
